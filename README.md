# odbmodel

extract model data from Abaqus odb files

## Usage

abaqus python src/odbmodel.py [-h] --odb ODBFILE [--instance INSTANCE] [--output OUTPUT]
                   [--postfunction POSTFUNCTION] [--interactive]
                   [--elset ELSETNAME | --nset NSETNAME | --nid NID | --eid EID]

### Prerequisites

Abaqus
